# priority-queue-cpp
# Priority Queue Implementation (C++)

## 📌 Description

This project implements a Priority Queue using Binary Heap in C++.

There are two versions:

* Max Heap (higher priority value = higher priority)
* Min Heap (lower priority value = higher priority)

## 🎓 Purpose

This project is part of my Data Structures course at Gia Dinh University.

## ⚙️ Features

* Push (insert element)
* Pop (remove highest priority element)
* Peek (view top element)
* Heapify Up & Down

## 📂 Files

* max_heap.cpp → Max Heap implementation
* min_heap.cpp → Min Heap implementation

## ▶️ How to run

```bash
g++ max_heap.cpp -o max
./max
```
